## Key Page Elements

### Navigation Anchors
- `#what-we-do` → How We Turn Attention Into Revenue section
- `#case-studies` → Case Studies section
- `#apply-section` → Primary CTA (Apply for Growth System)

### Design Tokens (Consistent Throughout)
- **Primary Color:** Teal (#14b8a6) — for accents, highlights, important CTAs
- **Background:** Dark gradient (#0a0e10 → #0f1318)
- **Typography:** Inter (400, 500, 600, 700, 800, 900)
- **Border Radius:** 20px (cards), 16px (sub-sections), 12px (buttons)
- **Spacing:** 120px sections, 60px padding, 24px cards

### Responsive Breakpoints
- **Desktop (1024px+):** 3-column grids, side-by-side layouts
- **Tablet (768px–1024px):** 2-column grids collapse to 1
- **Mobile (480px–768px):** Full-width single column
- **Small Mobile (<480px):** Optimized padding, smaller fonts

## Messaging Hierarchy

### Level 1 (Headline)
"Viral attention that converts to customers."

### Level 2 (Subheading)
"We build short-form content distribution systems + high-converting funnels that turn organic reach into measurable revenue."

### Level 3 (Supporting copy)
Explains each step: distribution → conversion → optimization

### Level 4 (Social proof)
Case studies with Problem/System/Result format

### Level 5 (Qualification)
"Who This Is For" section to filter leads

---

## Conversion Flow

1. **Awareness** → Hero section (gets attention)
2. **Interest** → "How We Turn Attention Into Revenue" (shows method)
3. **Consideration** → "What You Actually Receive" (clarifies offer)
4. **Trust** → Case Studies (proves it works)
5. **Qualification** → "Who This Is For" (self-select fit)
6. **Action** → CTAs (apply, see case studies)

## Performance Notes

- Page is static HTML (no frameworks)
- CSS is optimized with CSS variables (easy theme switching)
- Animations are GPU-accelerated (fade-in on scroll)
- Embedded YouTube videos (lazy load on click)
- Mobile-first responsive design
