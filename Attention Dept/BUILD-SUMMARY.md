## 🎨 Design Features

### Video Integration
- Full-width YouTube embeds (16:9 aspect ratio)
- Autoplay muted (desktop auto-plays, mobile respects browser defaults)
- Minimal, clean YouTube UI (modestbranding=1, rel=0)
- Metrics overlay positioned bottom-right with semi-transparent background
- Responsive metrics on mobile (switches to stacked layout)

### Metrics Overlay Style
- Semi-transparent dark background with backdrop blur
- Teal/cyan accent color (#14b8a6)
- Clean typography: label (small uppercase) + value (bold, teal)
- Responsive: adjusts positioning and sizing on mobile

### Philosophy Section
- 2-column grid layout (responsive to 1 column on mobile)
- Cards with hover lift effect
- Teal accent borders on hover
- Clean hierarchy: bold statement + supporting context

### Overall Theme
- **Dark mode default** (background: #0a0e10)
- **Teal/cyan accent color** (#14b8a6, #06d6a0)
- **High spacing** for luxury feel
- **Modern Inter font** with varied weights
- **Smooth scroll behavior** and fade-in animations
- **Glassmorphism elements** (backdrop blur, semi-transparent cards)
